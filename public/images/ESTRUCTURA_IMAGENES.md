## 🖼️ Tamaños Recomendados de Imágenes

### Por Tipo de Uso:

#### 1. **Grid / Galería (Thumbnail)**
- **Dimensión:** 400 x 400 px
- **Formato:** JPG o PNG
- **Peso:** 50-150 KB
- **Uso:** Página de categorías, listados de productos
- **Aspectoacio:** 1:1 (cuadrado)

#### 2. **Detalle Producto (Main Image)**
- **Dimensión:** 800 x 800 px
- **Formato:** JPG
- **Peso:** 150-300 KB
- **Uso:** Página de detalles del producto (principal)
- **Aspecto:** 1:1 (cuadrado)

#### 3. **Imágenes Secundarias (Detail Images)**
- **Dimensión:** 800 x 800 px
- **Formato:** JPG
- **Peso:** 150-300 KB
- **Uso:** Vistas adicionales en ProductDetail.tsx
- **Aspecto:** 1:1 (cuadrado)

#### 4. **Móvil Optimizado**
- **Dimensión:** 600 x 600 px
- **Formato:** JPG
- **Peso:** 80-150 KB
- **Responsive:** Se escala automáticamente

---

## ✅ Especificaciones Técnicas

### Formato de Imagen
- **Principal:** JPG (mejor compresión)
- **Alternativo:** WebP (más moderno, menor tamaño)
- **Evitar:** PNG para fotos (usa más datos)

### Compresión Recomendada
- **Calidad JPG:** 85-90%
- **Ratio compresión:** 1.2 - 1.5 MB por imagen original

### Nomenclatura en Código
Las imágenes se referencian así en `mockData.ts`:
```javascript
images: [
  '/images/music/vinyl/p1-01.jpg',
  '/images/music/vinyl/p1-02.jpg'
]
```

## 🔗 Integración con ProductDetail.tsx

El componente ProductDetail.tsx ya tiene soporte para múltiples imágenes:

```typescript
// Array de imágenes del producto
const [selectedImage, setSelectedImage] = useState(0)

// Se renderiza así:
<img src={product.images[selectedImage]} alt="Producto" />

// Puedes navegar entre imágenes
product.images.map((img, idx) => (
  <button onClick={() => setSelectedImage(idx)}>
    <img src={img} />
  </button>
))
```
